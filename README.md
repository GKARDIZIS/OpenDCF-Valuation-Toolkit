# OpenDCF: Transaction Services Valuation Toolkit 📊

OpenDCF is an open-source Python tool designed to democratize institutional-grade financial modeling. This project provides aspiring financial analysts, students, and investment clubs with an accessible framework for conducting Discounted Cash Flow (DCF) valuations without needing expensive proprietary software.

## The Mission
Bridging the gap between academic theory and real-world Transaction Services (M&A). OpenDCF is built to help users quickly assess Enterprise Value, calculate WACC, and project Free Cash Flows using standard Python libraries (`pandas`, `numpy`).

## Features
- **WACC Calculation**: Dynamic Weighted Average Cost of Capital modeling.
- **FCF Projection Engine**: Present Value calculations for unlevered free cash flows.
- **Terminal Value**: Gordon Growth Model integration.
- **M&A Screening**: Designed for quick, iterative valuation scenarios.
